# Snow Maiden vs Squirrels - Winter Battle Game

A classic Space Invaders-style game with a winter theme featuring the Snow Maiden fighting against an army of squirrels.

- **Snow Maiden Hero**: Control the Snow Maiden character at the bottom of the screen
